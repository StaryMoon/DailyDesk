## dev.to / Hashnode

Title: Building DailyDesk: a native macOS desktop planner with a tiny reward loop

DailyDesk is a Swift/AppKit project that turns a daily task list into a small desktop widget. v0.2 adds a local reward loop: completed tasks earn coins, coins unlock pet accessories, and a calendar panel shows recent completion history.

The interesting AppKit details are:

- a borderless accessory-style desktop window that still accepts first-click text input
- a pin mode that can float above windows or park at desktop-icon level
- local JSON state for task history and reward progress
- no background activity monitoring

GitHub: https://github.com/StaryMoon/DailyDesk
